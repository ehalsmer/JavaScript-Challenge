Javascript Assignment

The designer gave us the markup for a new page in design.html.

Create a dynamic version of this page, using lit-element. Clicking on a star should toggle between filled/unfilled. Clicking the button should add a new row with the value from the text field.

See the LitElement Guide for an introduction. Also feel free to use MDL Lite like the designer did.

Hint: modifying an array doesn't cause LitElement to re-render. Use this.data = newArray instead of this.data.push(...).
